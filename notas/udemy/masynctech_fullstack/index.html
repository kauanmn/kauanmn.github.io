Masynctech School - Full Stack Development

MERN Stack

  • Node.js: JavaScript runtime, runs JavaScript on servers, excels in scalability with its asynchronous, event-driven nature

  • Express.js: web app framework, it is a minimal Node.js framework for web apps, simplifying routing, API creation, and database integration

  • MongoDB: NoSQL database, MongoDB stores data in JSON-like format, is scalable, and provides an intuitive alternative to relational databases

  • React: frontend library, it is a popular JavaScript library for building user interfaces, emphasizing reusability and efficient UI updates through a virtual DOM

Why MERN stack?

  • Flexibility & scalability
  • JavaScript everywhere
  • Community support
  • Modern web development

HTML

  • HTML (HyperText Markup Language) is the foundation of web development
  • It is a markup language that uses tags to define the different elements on a web page
  • Created by Tim Berners-Lee, the creator of WWW, in 1993

Bootstrap

Bootstrap is a widely-used frontend framework released by Twitter in 2011, renowned for its ability to develop responsive and mobile-first websites, featuring a rich library of components, extensive documentation, and robust community support.

Use cases:

  • Rapid prototyping community
  • web applications
  • CMS (Content Management Systems)
  • educational projects and tutorials

Features

  • responsive design
  • mobile-first approach
  • rich component library
  • customizable
  • utility classes
  • preprocessor support
  • javascript plugins
  • accessibility
  • extensive documentation and community support
